# MyTetris
自作テトリス．基本的に動作は本家同様．
## How to use
以下のコマンドを入力すると自動的にアプリがビルドされ立ち上がる
```
cd MyTetris
make
```
ただし，libjfxwebkit.dylibを各自で入手しMytetris/libにダウンロードする必要がある．
## 操作
以下のコマンドでミノを操作できる．
```
A: 左, S: 下, D: 右, J:反時計回り, L: 時計回り
```

## Config
最初の設定画面でSettingに入ると，横のマス数と縦のマス数を変更できる．また，次に落ちてくるミノを非表示にもできる
